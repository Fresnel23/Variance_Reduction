# Variance Reduction for the MOS Method

Steady-state numerical simulations obtained with the **MOS (Monte Carlo-Oriented Solver)** using variance reduction techniques.

<p align="center">
  <i>Repository accompanying the development and validation of the MOS numerical methodology.</i>
</p>

---

## Overview

This repository contains steady-state animations illustrating the numerical solutions computed with the **MOS method** for representative benchmark geometries.

The animations are intended to provide a qualitative visualization of the convergence and final steady-state flow fields obtained with the proposed variance reduction strategy.
